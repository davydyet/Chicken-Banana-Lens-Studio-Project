# Chicken-Banana-Lens-Studio-Project
This is a fun chicken and banana animation, created using Lens Studio.

Features:
- Animated chicken and banana
- Synced with audio and controlled by facial expression and mouse click
- Fun visual effects (sunglasses)

Tools: Lens Studio, Blender

This project was created as part of my school assginment practice.
